2019 book list

Posted on 2019-12-30

今年看过的工作类图书

过去的一年工作重心不再偏向 iOS，所以也没有看任何跟 iOS 相关的书籍，走过程序员退休的年龄，之后会考虑向大前端（iOS & Android & Web）方向发展，同时兼顾 Java 后台路线，接受新的技术挑战。

《从零开始学架构》

介绍 Java 后台架构的一本书籍，真的能做到零基础也能对后台架构有一个较全面的认识。

前段时间做 Android 的朋友年底工作述职，被其他部门老大问到对后台技术高并发如何处理，我相信看过这本书的话，他也就不会抱怨为什么问移动客户端工程师后台的技术问题了，机会总是给有准备的人，你总是对现状不满，不知道如何努力就不努力，状态停滞不前又能怪谁没给你机会呢？看看不相关的技术也许对眼前的工作没有什么帮助，但是它能开阔视野和思路，今后工作中跟其他人沟通也不至于鸡同鸭讲。

《区块链——通往资产数字化之路》

这本书英文名字 Master Bitcoin 缩写 MB，MB 对于很多了解区块链的人来说不会陌生，豆瓣评分 9.0 。不管是从技术角度的剖析还是应用角度的讲解都非常全面到位，相对于《区块链革命》这本讲区块链会给人们生活带来什么改变的介绍书籍，通过 MB 你能够对区块链有一个比较具象的认识，而且书中介绍了加密算法、椭圆曲线加密算法、默克尔树等等对密码学的介绍也相当有趣。

《智能时代》

吴军博士的这本书虽然没怎么讲技术，但是读起来还算有趣，看他书写的思路也能看出来这位前辈的博学。

《大数据时代》

这种不讲技术但是又说技术的书，对我来说真的是吸引力不大，囫囵吞枣的看完，隔了一段时间以后对书中内容竟全无印象

2019 年看过的技术书籍还是太少了，主要是工作繁忙还有这两本书啃得比较费劲，对于书中很多知识的不理解，需要时间理解消化，不找借口了继续努力吧。

管理类书籍

今年竟然没有读多少管理类书籍，甚是可怕，看了很多似技术又非技术的书籍，就算是为了管理工作做的铺垫吧

《赋能》

算是今年阅读的少数管理类书籍中的一本，讲美军海豹突击队的管理，有一定的启发性。

《目标》

艾利·高德拉特的作品是教授推荐的书，对项目管理等有一个重新的认识，看过本书后在工作中出现人手不足的问题时，我总是会提醒自己哪里是瓶颈，是否能够优化。

《阿米巴经营》

年初朋友去日本说在京瓷旁边参观，问我听说过京瓷么，微微一笑那时正在看稻盛和夫的作品，怎会不知。阿米巴是否适合中国，看看海尔今后的发展再说。

《边界——企业机会出现在组织边界被打破的地方》

美国越来越重视打破边界，不知道企业中会不会出现新的组织结构，但是绝对是个趋势。

非工作类书籍推荐

《基因传》

一本非常浅显易懂介绍基因医学史的书，从中学课本中的孟德尔豌豆杂交到达尔文进化论，再到基因的发现很多有意思的故事，看完本书对事物背后的原理故事更感兴趣，突然想重读《别逗了，费曼先生》，作为女生之前对这种书籍真的是硬着头皮看完，还找不到任何感兴趣的点，看完本书重燃了兴趣。
